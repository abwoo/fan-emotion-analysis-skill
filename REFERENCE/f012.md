# 女性分类速查 - 结构化调用版

## 信息处理索引

- **分类对象**：国女、雌性女、独立女性、精神小妹。
- **核心用途**：先判断对方的择偶框架，再决定用画饼、硬实力、精神共鸣还是快速推进。
- **关键输入**：职业、颜值段位、朋友圈风格、消费水平、茶艺程度、是否缺爱、是否看重长期关系。
- **关键输出**：女性类型、核心需求、无效打法、推荐策略、止损条件。

## 类型判断表

| 类型 | 主要特征 | 核心需求 | 推荐策略 |
| --- | --- | --- | --- |
| 国女 | 传统路径、现实感弱、幻想强 | 不国男、高阶层、对她好、长择 | 配合幻想、画饼、建立高价值长期叙事 |
| 雌性女 | 靠脸吃饭、茶艺强、颜值变现 | 钱、资源、长期大额付出 | 判断交换性价比，普通男性慎入 |
| 独立女性 | 事业自立、追求平等、精神要求高 | 同阶层、共鸣、尊重 | 攻心、深聊、展现理解与内涵 |
| 精神小妹 | 成长复杂、江湖化、反馈直接 | 情绪陪伴、社会气、适度小付出 | 快速线下推进，做好边界和健康防护 |

## 调用规则

1. 如果用户只给聊天记录，先从回复内容、主动性、消费暗示和朋友圈信息推断类型。
2. 如果类型不确定，最多补问 3 个信息：职业/朋友圈风格/消费与恋爱观。
3. 类型判断完成后，再调用聊天、邀约、约会或长期关系分卷。

核心心法：先快速判断女生类型，再用对应策略，无效则及时止损。

1. 国女
特征：成长于传统路径（好学生→好工作），现实感弱，恋爱靠“许愿”，对男性有脱离现实的幻想（“既要又要”）。颜值通常≤6分，茶艺弱。
择偶框架（优先级）：
不国男（不被动、会包装）+ 高阶层 + 对她好 + 长择
若不“对她好”，则需 “帅”（让她心动）替代。
若两者都无，必须会 “画饼”（描绘美好未来）。
策略：别讲道理，配合演出。扮演她幻想中那个高价值、宠爱她、给她承诺的男主角。
2. 雌性女（茶艺/靠脸吃饭）
特征：以美貌为核心生产资料变现（如主播、混圈女）。颜值(5.8-7分)，茶艺中等偏上，多出身普通。
择偶框架（优先级）：
不丑 + 愿意大额、长期付出（钱/资源） + 愿意娶她（终极目标）。
次选：长期大额供养。
帅可降低金钱要求，但仍需基础付出。
策略：硬实力（财/貌/阶）交换。真诚、聊天技巧等无效。普通男性慎入，易人财两空。
3. 独立女性
特征：事业成功靠自身能力（女老板、高管等），经济精神独立，追求平等，厌恶大男子主义。颜值中等(4.5-5.8分)，茶艺中等或弱。
择偶框架（优先级）：
同一阶层（思维同频） + 形象过关 + 精神共鸣（核心） + 长择。
帅可弥补部分阶层差距。
策略：攻心为上。展现内涵、深度沟通、理解与尊重。巨额物质付出性价比低，真诚的理解更打动她。
4. 精神小妹（补充）
特征：成长环境复杂（缺爱、边缘化），从事边缘工作，价值观“江湖化”，及时行乐。是情感实践的“入门对象”。
择偶偏好：接受“社会气”的帅（黄毛属性）、高情绪陪伴、适度小付出（反馈直接）。看不起传统“国男”。
策略与风险：
策略：快速线下推进（打台球、吃烧烤），高强度情绪陪伴，适度小付出。
风险：必须做好健康防护，并进行框架测试（树立底线，筛选可塑者），防止被过度索取或利用。
