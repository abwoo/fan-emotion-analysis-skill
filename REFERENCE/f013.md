# 女性颜值评分 - 结构化调用版

## 信息处理索引

- **评分范围**：4 分到 9.5 分。
- **核心用途**：判断目标稀缺度、竞争强度、用户当前条件是否匹配。
- **关键输入**：素颜/妆后差异、现实稀缺度、社交软件表现、是否明星/网红级、用户自身形象和阶层。
- **关键输出**：颜值分段、追求难度、普通男性可行性、需要的优势类型。

## 调用规则

1. 先区分“照片分”和“现实分”，避免被妆造、滤镜、角度误导。
2. 7 分以上默认进入高竞争区，需要结合 `top-beauty-guide.md` 判断类型和窗口。
3. 6.5 分以下可进一步结合 `female-classification.md` 判断策略，不单靠颜值给结论。
4. 如果用户自身条件明显低于目标分段，必须指出难度，不给虚假鼓励。
