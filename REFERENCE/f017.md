# 男性分类女性版 - 结构化调用版

## 信息处理索引

- **二维坐标**：魅力值 × 物质条件与付出意愿。
- **核心用途**：从女性视角反推用户在对方心中的位置，是短择、长择、备选、资源位还是体验位。
- **关键输入**：用户形象与情绪价值、经济条件、付出意愿、女方反馈、是否主动、是否索取、是否愿意单独见面。
- **关键输出**：女性视角定位、吸引来源、被卡原因、应补的筹码。

## 九宫格调用方式

| 判断维度 | 需要观察的问题 |
| --- | --- |
| 魅力值 | 她是否主动、是否开心、是否愿意延长互动、是否有身体/情绪靠近 |
| 物质条件 | 她是否认可你的阶层、工作、生活方式和未来性 |
| 付出意愿 | 她是否测试你愿不愿意投入钱、时间、资源或承诺 |

## 调用规则

1. 当用户反馈“她忽冷忽热、愿意收好处但不推进、只聊天不见面”时优先读取。
2. 不从男性自我感觉出发，要从女方行为反推用户在她眼中的类型。
3. 若魅力低但付出高，提醒资源位风险；若魅力高但条件低，提醒体验位和未来性不足。

**择偶两大核心：**

1. **魅力值** ：让他“上头”的感觉。帅、有趣、懂你，提供情绪体验。
2. **物质条件与付出意愿** ：他能给你的实际好处。钱、资源、人脉、地位光环。

**核心洞见：**

* 几乎没有男人能两者都顶级（传说级不存在）。
* 魅力值高的男人，通常不缺女生，反而不太愿意花钱。

| `` | 物质条件低         | 物质条件中         | 物质条件高                        |

| 魅力值低  | 类型一：双低男     | 类型二：传统国男   | 类型三：油腻付出男                |

* **双低男** （魅力低、条件低）：没感觉也没钱。**策略：远离。**
* **传统国男** （魅力低、条件中）：老实人，稳定但无趣。付出流程化。**策略：展示宜家宜室，寻求婚姻。**

**【中坚型】—— 常见婚恋对象**

* **趣味小能手** （魅力中、条件低）：有趣、有才，但穷且没规划。**策略：仅供体验，别图未来。适合女强人当调剂。**

* **极致体验家** （魅力高、条件低）：情场高手，提供极致情绪体验，目标是从你这搞钱。**策略：极度清醒，守住钱包，仅当体验或消费。**

**给你三句心法：**

1. **快速判断** ：先感受相处是否开心（魅力），再看他有啥并愿否给你（物质）。
2. **动态眼光** ：人和关系都会变，用发展的眼光看。
