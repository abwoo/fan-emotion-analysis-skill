# 视频调用层（增强版）

## 内容定位

- 形式：街头实战剪辑 + 自述
- 立意：用最不利的"外卖员"身份在杭州做即时邀约挑战

- 段一：开场对白片段（"先告诉我你多大""89K吧""门当户对")
