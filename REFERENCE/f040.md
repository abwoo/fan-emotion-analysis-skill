# 视频调用层（增强版）

## 内容定位

- 形式：主播 + 多位女嘉宾访谈
- 立意：从女嘉宾原话还原她们对"难忘男生"的真实判断
- 用户最常问：女生到底看重什么样的付出、装穷会不会有效、家境差但用心有用吗

## 主题锚点（按访谈顺序）

- 段一：第一个话题——最让你难忘的男生做了什么
- 段二：兔兔回答——家境不好却倾尽所有买礼物（"反而很感动"）
- 段三：主播提问——如果他有钱却装作没钱怎么办（"这不可以 不要被我知道"）
- 段四：CC 回答——出国前的难忘故事
- 段五：后续话题（让你下头的男生 / 筛选体系等扩展）

- "他為了展錢給我買禮物"
