# 一约得吃-机制总表

## 说明

- 本表按研究视角汇总材料中的核心机制链，不作为执行指南，而是作为文本结构分析工具。

## 机制链条

### 对象筛选机制

- 链条描述：先按颜值、风格、可得性、情欲窗口、经济属性和文化属性分类，再决定是否进入后续流程。
- 研究含义：把亲密互动预先量化为值得不值得投入的筛选题。
- 标签：`等级筛选 / 对象化叙事`

### 高价值包装机制

- 链条描述：通过身高、长相、穿搭、照片、留学、朋友圈、社交货币塑造产品化自我。
- 研究含义：让对方在见面前先接受既定身份故事。
- 标签：`身份包装 / 伪自然化`

### 窗口识别机制

- 链条描述：持续寻找分手、空窗、孤独、婚姻受挫、深夜、无聊等时机。
- 研究含义：将脆弱与需求直接编码为可推进窗口。
- 标签：`脆弱期切入`

### 酒精增幅机制

- 链条描述：把喝酒描述为同时提升兴奋、降低顾虑、加快亲密升级的工具。

- 标签：`酒精推进`

- 链条描述：从基础互动到隐私、情感、性话题，再到打分和阈值问题。
