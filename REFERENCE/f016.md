# 女性出轨风险识别 - 结构化调用版

## 信息处理索引

- **风险分档**：必出轨、高概率、一半概率、潜在风险。
- **核心原则**：观察可验证行为，不以口头解释作为主要证据。
- **关键输入**：穿着变化、社交软件、异性边界、失联频率、手机保护、消费变化、亲密排斥。
- **关键输出**：风险档位、证据强度、是否继续观察、是否止损。

## 风险判断流程

1. 先收集客观行为，不凭情绪和猜测定性。
2. 命中第一、二档多项特征时，优先保护自己，减少争辩和投入。
3. 第三、四档只能作为观察信号，需要结合长期关系质量与近期变化。
4. 输出时区分“证据”“推测”“建议动作”。

**核心思路：** 观察可验证的客观行为，而非听信言辞。风险由高到低分为四档。

**一、【必出轨】满足即终结**

* **核心结论：** 立即远离。

  4. **思想肤浅：** 交流仅限于物质、八卦，缺乏深度和自律。

  6. **道德感低下：** 有过劈腿史且合理化，无忠诚观念。

**二、【高概率出轨】高度危险**

* **核心结论：** 关系已极度危险，需立刻审视。
* **关键依据：**

  1. **沉迷社交软件：** 频繁使用探探等约会软件。
  2. **异性朋友无边界：** 多名“男闺蜜”，互动亲昵，并打压你的正常质疑。
  3. **行踪诡秘：** 频繁失联、以“闺蜜”为由高频外出、外出频率突增。
  4. **过度保护手机：** 永远反扣、洗澡携带、查看则激烈反抗。
  5. **沟通异常：** 接电话躲闪、眼神轻浮挑逗。
  6. **消费突变：** 突然购买大量香水、性感内衣等取悦他人的私密物品。

**三、【一半概率】悬崖边缘**

* **关键依据：**

  1. **情感身体排斥：** 拒绝亲密接触，性生活锐减。
  2. **无故指责你：** 突然频繁挑剔你的缺点，为潜在不忠找理由。
  3. **情绪随信息波动：** 独处时因手机信息窃喜或焦虑。
  4. **外貌管理突变：** 突然疯狂健身、改变为“媚男”风格打扮。
  5. **性边界模糊：** 可与陌生人开黄腔，接受**速推**（短期发生关系）。
  6. **出身贫寒但物欲强：** 将物质视为首要目标，无法忍受生活水平下降。

**四、【潜在风险】易感体质**

* **核心结论：** 具备出轨的易感条件，需保持观察。

  1. 轻易接受马路搭讪并给联系方式。

  3. 热衷参加各类复杂社交活动，增加接触诱惑的机会。
  4. 隐瞒新结识的异性朋友。
  5. 好吃懒做，无正经事业，精神空虚。 *（注：面相特征主观性强，仅供参考，不列为主要依据）*

**【核心心态与行动指南】**

* **放弃幻想，接受现实：** 你不是在做学术研究，而是在保护自己。不要侥幸认为“她会是个例外”。
* **立即止损，实用主义：** 一旦发现对方符合第一、二梯队多项特征，或叠加大量第三、四梯队特征，最明智的做法是果断离开，停止内耗。
* **相信观察，而非辩解：** 信任你看到的客观迹象和直觉警报。
* **定期自查：** 将此指南作为“安全检查表”对照伴侣。预防远优于治疗。
