## 五、调用优先级（视频 vs 文字 references）

读取顺序按"用户诉求性质"决定：

- **要原话 / 直播语气 / 视频里怎么说** → **优先视频文件**，文字 references 兜底
- **要可执行话术 / 流程 / 模板** → **优先文字 references**，视频补充语感
- **要诊断与判断（罪项、类型、阶段）** → **视频 + 文字并读**，先视频做证据，再文字给方案
- **要止损或风险判断** → **文字 references 优先**（`female-cheat-risk.md` / `laonv.md`），视频做佐证
