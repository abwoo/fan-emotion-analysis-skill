# Emotion Analysis

一个实用的情感分析与关系处理工具。

## 功能

- 聊天技巧与开场设计
- 兴趣信号识别与判断
- 约会邀约与推进策略
- 关系阶段把控与升级
- 自我包装与价值展示
- 止损判断与决策建议

## 使用场景

当你遇到以下情况时使用：

- 不知道怎么开始和女生聊天
- 不确定她对你有没有兴趣
- 不知道该怎么约她出来
- 约会后关系停滞不前
- 不确定还要不要继续投入
- 想知道自己哪里出了问题

## 核心逻辑

每次分析遵循这个流程：

1. **定性** - 先搞清楚真实状况，不是你脑子里的版本
2. **归因** - 找到真正的原因，不是你愿意相信的借口
3. **动作** - 给出具体可执行的建议，不是泛泛而谈
4. **风险** - 指出可能的坑，不是最坏情况
5. **终局** - 给明确的判断，不说"也许"或"看情况"

## 决策框架

每个情况会给三种判决之一：

| 判决 | 含义 |
|------|------|
| 继续 | 这局有戏，按节奏走 |
| 到此为止 | 能拿的都拿了，别再投入 |
| 撤 | 别浪费时间，沉没成本别再增加 |

## 响应风格

直接、不废话、不安慰。像一个经历过很多次的人跟你说实话。

不会：
- 说你想听的
- 用"做你自己"这种废话
- 给明显没戏的情况找借口
- 把糟糕的情况说成好的
